Requisitos Técnicos para firmar documentos en el Portafirmas

Requisitos comunes

  • Disponer de un Certificado digital aceptado por el Portafirmas (Camerfirma, FNMT o DNIe) y tenerlo instalado en el equipo/dispositivo.

Requisitos para equipos de escritorio

  • Navegador: Internet Explorer 9 en adelante, Mozilla Firefox, Google Chrome.
  • En caso de usar Windows 8, 8.1 o 10, es necesario ejecutar el navegador fuera del modo "metro".
  • Tener instalada en el ordenador la aplicación de Autofirma. Para descargar el Cliente de Autofirma, haga click aquí.

Requisitos para dispositivos móviles

Dispositivos Android

  • Navegador: Navegador por defecto a partir de la versión 4.0 de Android, Google Chrome, Mozilla Firefox.
  • Tener instalado el Cliente Firma Móvil

Dispositivos iOS (Apple)

  • Navegador: Safari.
  • Tener instalado el Cliente Firma Móvil

Para cualquier duda por favor contacte con el área de Sistemas de la Autoridad Portuaria de Santa Cruz de Tenerife.
